Requesting Letters of Recommendation


I receive many requests each semester for letters of recommendation. Please follow the instructions below to help me write a good letter and ensure that I can meet any deadlines.

  1. Fill in the Stony Brook Student Reference Request Form to authorize release of information in the letter and hand it to me.
  2. E-mail me your current CV/resume and a paragraph describing your interests and career goals, and anything else I should know to help write a good letter.
  3. Remind me what classes you took with me (and what semester).
  4. Create a google docs spreadsheet with columns for:
    1. program name
    2. institution
    3. URL
    4. instructions on how I send the letter
    5. reason you are interested in the program
    6. due date
    7. status
    The last column is for me to use to indicate if I've sent the letter. Please sort these in order of due date (most urgent at the top of the list). Be sure to share the doc with me and grant me write access.

Please request letters no later than 3 weeks before the first due date
