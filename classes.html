Teaching Resources

  • Introductory Astronomy Animations

    In support of my classes, I make some simple introductory astronomy animations available.

  • Computational Astrophysical Hydrodynamics text


    For students working with me, I've written a text Introduction to Computational Astrophysical Hydrodynamics. The LaTeX source for these notes are available on github. If you want to contribute, fork it, edit, and issue a pull request.

  • Open Astrophysics Bookshelf

    The Open Astrophysics Bookshelf is a project that hosts open-licensed textbooks that can be collaboratively edited, crowdsourced, etc.

  • Python Boot Camp

    A one-week python boot camp for summer REU students.

  • Teaching and Learning with Jupyter

    Teaching and Learning with Jupyter is a handbook with best practices for teaching with Jupyter.
